## 一、项目范围

本阶段**仅开发管理系统**，包括：

| 模块 | 优先级 | 说明 |
|------|--------|------|
| 仪表盘 | P1 | 系统指标、统计信息、OpenClaw 状态监控、Token 使用 |
| 人类账号管理 | P1 | CRUD + 启用/禁用 + 自动解雇逻辑 |
| OpenClaw 管理 | P1 | CRUD + 启用/禁用 + 同事发消息权限 + 工作空间管理 |
| 雇佣关系管理 | P1 | 查看全局雇佣关系 + 历史记录 |
| 系统日志 | P1 | 操作日志查询（保留 2 个月） |
| 角色管理 | P2 | 预制角色模板管理 |
| 记忆管理 | P2 | OpenClaw 记忆数据管理 |
| 系统设置 | P2 | 系统参数配置（含雇佣数量上限） |

## 二、技术栈

### 2.1 后端

| 组件 | 技术选型 | 说明 |
|------|----------|------|
| 框架 | FastAPI | 异步支持、自动生成 OpenAPI 文档 |
| ORM | SQLAlchemy 2.0 | 类型提示完善、支持异步 |
| 迁移 | Alembic | 数据库版本管理 |
| 认证 | JWT + bcrypt | 管理员登录认证 |
| 数据库 | PostgreSQL | 主数据库 |
| 日志 | Python logging + 数据库 | 操作日志持久化 |

### 2.2 前端

| 组件 | 技术选型 | 说明 |
|------|----------|------|
| 框架 | Vue 3 | Composition API |
| 构建工具 | Vite | 快速开发、热更新 |
| 语言 | TypeScript | 类型安全 |
| UI 组件库 | Element Plus | 丰富的企业级组件 |
| 状态管理 | Pinia | Vue 3 官方推荐 |
| 路由 | Vue Router 4 | 单页应用路由 |
| HTTP 客户端 | Axios | 请求拦截、响应处理 |

### 2.3 部署

| 组件 | 方式 | 说明 |
|------|------|------|
| 后端 | 直接部署 / Docker | 按开发框架规范 |
| 前端 | Nginx 静态托管 | 构建后部署 |
| 数据库 | PostgreSQL | 独立服务 |

## 五、关键设计决策

| 决策点 | 选择 | 理由 |
|--------|------|------|
| 雇佣数量上限 | 可配置，默认 3 | 灵活性 + 合理限制 |
| 管理员认证 | JWT + 用户名密码 | 简单、成熟、易实现 |
| 密码加密 | bcrypt | 业界标准、安全可靠 |
| 日志保留 | 2 个月 | 平衡审计需求与存储成本 |
| UI 组件库 | Element Plus | Vue 3 生态完善、文档好 |
