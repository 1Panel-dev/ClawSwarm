## 🔧 技术栈汇总

### 后端
- **框架**: FastAPI
- **ORM**: SQLAlchemy 2.0
- **迁移**: Alembic
- **认证**: JWT + bcrypt
- **数据库**: PostgreSQL 14+

### 前端
- **框架**: Vue 3
- **构建**: Vite
- **语言**: TypeScript
- **UI**: Element Plus
- **状态**: Pinia
- **路由**: Vue Router 4
